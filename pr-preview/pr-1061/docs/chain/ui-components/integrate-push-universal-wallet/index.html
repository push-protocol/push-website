Integrate Push Universal Wallet

Overview​

Seamlessly Connect to Multiple Blockchains: Simplify wallet integration across your application with Push Universal Wallet.

You can effortlessly integrate the Push Universal Wallet into your root application component, enabling wallet functionality throughout your entire app. This eliminates the need to manage complex blockchain connections, authentication states, and cross-chain interactions manually.

How It Works​

The Push Universal Wallet operates on a simple provider-consumer pattern:

  1. Provider Setup: Wrap your application with PushUniversalWalletProvider to initialize wallet functionality
  2. Button Integration: Add PushUniversalAccountButton components where users should connect
  3. State Management: Use the usePushWalletContext hook to access wallet state and methods throughout your app

The Push Universal Wallet provides a unified interface for connecting to multiple blockchains, automatically handling authentication and cross-chain interactions without complex blockchain-specific implementations.

Installation​

  • npm
  • yarn
npm install @pushchain/ui-kit
yarn add @pushchain/ui-kit

Usage and Playground​


// Import necessary components from @pushchain/ui-kit
import {
  PushUniversalWalletProvider,
  PushUniversalAccountButton,
  CONSTANTS,
} from '@pushchain/ui-kit';

function App() {
  // Define Wallet Config
  const walletConfig = {
    env: PushChain.CONSTANTS.ENV.DEVNET,
    login: {
      email: true,
      google: true,
      wallet: {
        enabled: true,
      },
      appPreview: true,
    },
    modal: {
      loginLayout: PushChain.CONSTANTS.LOGIN.SPLIT,
      connectedLayout: PushChain.CONSTANTS.CONNECTED.HOVER,
      appPreview: true,
    },
  };

  // Define Your App Preview
  const appMetadata = {
    logoUrl:
      'https://plus.unsplash.com/premium_photo-1746731481770-08b2f71661d0?q=80&w=2671&auto=format&fit=crop&ixlib=rb-4.1.0&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D',
    title: 'Test App Title',
    description: 'Test App Description',
  };

  return (
    <PushUniversalWalletProvider config={walletConfig} app={appMetadata}>
      <PushUniversalAccountButton />
    </PushUniversalWalletProvider>
  );
}

Customization Parameters​

Customize the parameters as per your need and the wallet functionality being used.

PushUniversalWalletProvider​

ParamTypeDefaultRemarks
configPushUniversalWalletProviderConfig-Core configuration object for wallet connections
config.envPushChain.CONSTANTS.ENV-Chain environment - 'TESTNET', 'MAINNET', or 'LOCAL'
config.uidstring'default'Unique identifier for this provider instance
config.rpcURLstring-Custom JSON-RPC endpoint for supported chains
config.loginObject-Login method configuration (email, google, wallet)
modalModalProps-Global defaults for login and connected modal instances
appAppMetadata-Application branding metadata (logo, title, description)
themeModePushChain.CONSTANTS.THEMEPushChain.CONSTANTS.THEME.LIGHTTheme mode - 'LIGHT' or 'DARK'
themeOverridesThemeOverrides-Override the theme settings
childrenReact.ReactNode-Your application components

Note: Parameters in this style are mandatory.

PushUniversalAccountButton​

ParamTypeDefaultRemarks
uidstring-Target provider instance, matches config.uid
connectButtonTextstring'Connect Account'Text for the connect button
loadingComponentReact.ReactNode-Custom loading indicator
modalAppOverrideAppMetadata-Override the modal app metadata
loginAppOverrideAppMetadata-Override the login app metadata
themeOverridesButtonThemeOverrides-Override the button theme settings

Note: Parameters in this style are mandatory.
