Intro to Push Chain

Push Chain is the first True Universal Layer 1 blockchain. It is a 100% EVM compatible, proof of stake(PoS) chain built on top of the Cosmos SDK.

It enables applications to be built once and used by users across multiple blockchains (including non-EVM chains). This allows developers to create a single universal application that is unlimited in interoperability across all chains.

Under the hood, it utilizes wallet abstraction and universal fee abstraction to enable users of any chain (Etherem, Solana, etc) to use the same app deployed on Push Chain.

Push Chain Innovations​

Push Chain is designed to abstract away the complexity of blockchain fragmentation.

It introduces several innovations to abstract away developer's pain points, allowing them to multiply their userbase manifold by enabling compatibility with several L1s. Innovations include:

  • Zero EVM Code Changes: Developers can deploy their EVM smart contract with zero changes to enable universal interoperability.
  • Wallet Abstraction: Enables any wallet (Metamask, Phantom, etc) to be compatible with apps on Push Chain. Also allows email or social login for normies.
  • Universal Fee Abstraction: Enables users of any chain to fund transactions natively from any other chain. For instance: ETH users can fund their tx using native ETH balance or any ERC-20 balances without the need to switch networks. This is true for Solana users or for any other supported chains.
  • True Mapping and Attribution of Txs across chains: Each transaction done by users of different chains are mapped and attributed, enabling shared app experience among users of all chains.
  • Universal Smart Contract: Enables smart contract of Push Chain to recognize users of other chains for any arbitary logic.
  • Shared State and Programmable Solvers: Future features that will also enable reading from other chains and writing to other chains.

Why Build on Push Chain?​

Developers should build on Push Chain for the following reasons:

  • Expand your userbase instantly: If you have an EVM app, you can deploy it on Push Chain with zero changes to enable universal interoperability. non-EVM apps are supported as well.
  • Deploy without audits or friction: No changes in your solidity codebase needed, just deploy and use our wrapped SDKs to enable universal access.
  • Smoother, unfragmented user experience: Users now use a single app no matter what chain they come from, creating apps that are accessible from any wallet on any chain.
  • Fee and Wallet Abstraction: Fees can be paid in native tokens or handled by applications, removing friction for users.
  • Comprehensive Wallet Abstraction: Simplifies onboarding for Web3 enthusiasts, newcomers, and traditional Web2 users.

Experience Push Chain​

The best way to see Push Chain in action is by interacting with it. You can:

  1. Visit simulate.push.org to interact directly. You can log in with your existing wallet, email, or even your social accounts, and send a transaction on Push Chain with just a few clicks.
    https://simulate.push.org
  2. Alternatively, use our live playgrounds to experiment with code and see live updates within our documentation.
  3. Learn more about Push Chain fundamentals, how it works, deep dives, and developer resources in our comprehensive Knowledge Base.
