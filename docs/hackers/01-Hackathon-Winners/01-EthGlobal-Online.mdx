# EthGlobal Online 2023

[ETHOnline 2023](https://ethglobal.com/events/ethonline2023), organized by ETHGlobal, is a key event in the web3 realm, combining a hackathon with over $200,000 USD in prizes and summits to showcase the latest in web3. Push Protocol proudly sponsored **$5,000** for projects that innovatively integrated its decentralized communication network, supporting the vibrant community of developers.

<h2 align="center">Winners</h2>

### _ICE Alerts_

ICE Alerts is a decentralized app that uses NFC technology to provide crucial medical information through an ICE Medical Alert Card or Bracelet, ensuring accessibility in emergencies even when the individual is non-communicative, by combining blockchain's security with NFC's ease of access.

### _Gated Bounty Protocol (GBP)_

The Gated Bounty Protocol (GBP) is a secure bounty system utilizing soulbound NFTs and reputation tokens for user verification. It features smart contracts for flexible bounty management and a React-based frontend integrating Push Protocol for wallet notifications, gated chats, and dedicated communication spaces for bounties.

### _ETH-Line_

This project enhances web3 connectivity by offering video and voice calls, along with chats without traditional logins, using biometrics and passcodes for authentication. It supports chat through Push Protocol and XMTP, leverages WebRTC for calls, and uses regex parsing for notification intent detection.
