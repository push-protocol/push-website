# Intro to Push Chain

Push Chain is a Proof of Stake (PoS) layer 1 blockchain network designed to revolutionize web3 by addressing scalability, interoperability, and user experience challenges. With a roadmap spanning three phases, Push Chain aims to empower developers to create universal applications that blend financial and non-financial use cases, making blockchain accessible to everyone.

## Why Push Chain?

Push Chain focuses on consumer-centric applications with seamless UX, scalability, and simplified interactions. Unlike finance-focused blockchains, it enables unique innovations like parallel block execution and dynamic sharding, enhancing usability. Push Chain enables:

- **Empowering Consumer Applications**: Supports financial and non-financial use cases with seamless cross-chain functionality.
- **Wallet Interactions Made Easy**: Users can interact using existing wallets without creating a new Push Wallet.
- **Fee Abstraction for User Convenience**: Transaction fees can be paid in native tokens or covered by applications for a seamless experience.
-	**Unified User Identity (Push ID)**: Links multiple wallets under one identifier for a unified view of assets and activities.
-	**Comprehensive Wallet Abstraction**: Provides easy onboarding for Web3 users, newcomers, and Web2 users alike.

## How Push Chain works?

All communication with Push Chain is signed and validated by a set of nodes, including validator nodes for transaction verification, storage nodes for efficient data handling, and archival nodes for maintaining complete transaction history. Read the <AImp href="https://whitepaper.push.org/" title="Push Chain Whitepaper">Push Chain Whitepaper</AImp> for an overview of its architecture, node roles, and decentralization efforts.

## Try Push Chain

Best way to understand Push Chain is to interact with it directly. To get started —

1. Head to [https://simulate.push.org/](https://simulate.push.org/)  
2. Login with your **email** or **existing wallet**. Push Chain supports **multiple login options** including —  

  - **Email Login** — Use your email for quick and easy onboarding.  
  - **Wallet Addresses** — Connect with wallets like MetaMask or any compatible Web3 wallet.  
3. Submit a transaction to the Push Chain.  
4. View your transaction live on **Push Scan**, the blockchain explorer for Push Chain: [https://scan.push.org/transactions](https://scan.push.org/transactions).
