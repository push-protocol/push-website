# Send Transaction Overview

Push Chain is an **EVM-compatible Universal Blockchain** that offers multiple ways to send transactions. The preferred method is using the Push Chain SDK for its advanced features, but you can also interact with Push Chain using standard EVM libraries such as `Viem` or `Ethers.js`.

## Send Transaction API

```typescript
const txHash = await pushChainClient.universal.sendTransaction({
  to: '0xa54E96d3fB93BD9f6cCEf87c2170aEdB1D47E1cF',
  value: BigInt('100000000000000000'), // 0.1 ETH in wei
});
```

### Send Transaction Parameters

| **Arguments** | **Type** | **Default** | **Description**                                       |
| ---------------------- | -------- | ----------- | ----------------------------------------------------- |
| _`to`_                 | `string` | -           | The address of the recipient.                         |
| _`value`_              | `BigInt` | -           | The value to send in wei.                             |
| `data`                 | `string` | -           | The data to send.                                     |
| `gasLimit`             | `BigInt` | -           | The gas limit for the transaction.                    |
| `maxFeePerGas`         | `BigInt` | -           | The maximum fee per gas for the transaction.          |
| `maxPriorityFeePerGas` | `BigInt` | -           | The maximum priority fee per gas for the transaction. |
| `deadline`             | `BigInt` | -           | The deadline for the transaction.                     |

> Note: These _`Arguments`_ are mandatory.

### Send Transaction with Contract Interaction

When you need to interact with smart contracts, you'll need to pass encoded function data. Here's how to generate the `data` parameter using viem and use it with the Push Chain SDK:

<Tabs groupId="send-transaction">
  <TabItem value='ethers_v6' label='Ethers (v6)' default>

```typescript
import { ethers } from 'ethers';

async function main() {
  // Define the ABI for the ERC20 transfer function
  const erc20Abi = [
  'function transfer(address to, uint256 amount) returns (bool)',
  ];

  // Create an Interface instance
  const interface = new ethers.Interface(erc20Abi);

  // Encode the function data
  const data = interface.encodeFunctionData('transfer', [
  '0xRecipientAddress',
  ethers.parseEther('10'),
  ]);

  // Send the transaction using Push Chain SDK
  const txHash = await pushChainClient.universal.sendTransaction({
    to: '0xTokenContractAddress', // The smart contract address on Push Chain
    value: 0n, // No ETH being sent
    data: data, // Encoded function call
  });
}

main().catch(console.error);
```

  </TabItem>

  <TabItem value='viem' label='Viem'>

```typescript
import { encodeFunctionData, parseEther } from 'viem';

async function main() {
  // Define the ABI for the ERC20 transfer function
  const erc20Abi = [
    'function transfer(address to, uint256 amount) returns (bool)',
  ];

  // Generate the encoded function data using viem
  const data = encodeFunctionData({
    abi: erc20Abi,
    functionName: 'transfer',
    args: ['0xRecipientAddress', parseEther('10')] // Transfer 10 tokens
  });

  // Send the transaction using Push Chain SDK
  const txHash = await pushChainClient.universal.sendTransaction({
    to: '0xTokenContractAddress', // The smart contract address on Push Chain
    value: BigInt('0'), // No ETH being sent, just contract interaction
    data: data // The encoded function call
  });
}

main().catch(console.error);
```

import { createWalletClient, http, parseEther } from 'viem';
import { privateKeyToAccount } from 'viem/accounts';
import { PushChain } from '@pushchain/core';

async function main() {
  const privateKey = 'YOUR_PRIVATE_KEY';
  const account = privateKeyToAccount(privateKey);

  const walletClient = createWalletClient({
    account,
    chain: PushChain.CONSTANTS.VIEM_PUSH_TESTNET,
    transport: http(),
  });

  const txHash = await walletClient.sendTransaction({
    to: '0xRecipientAddressHere',
    value: parseEther('0.1'),
  });
}

main().catch(console.error);
