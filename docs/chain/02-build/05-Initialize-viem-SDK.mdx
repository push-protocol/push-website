# Initialize Viem SDK Overview

The Push Chain SDK exports a bundled version of `viem`, which you can optionally import directly from `@pushchain/core`.

> Note: While this allows you to skip manually installing `viem` and `defineChain` yourself, you can also use your own viem installation if you prefer. The bundled version is provided as a convenience, but it's not mandatory to use it.

If you need more information about how to use `viem`, please refer to the [official viem documentation](https://viem.sh/).

## Initializing Viem Client

The Push Chain SDK exports a bundled version of `viem`, so you can import it directly from `@pushchain/core`. This allows you to skip manually installing `viem` and `defineChain` yourselfâ€”everything you need comes from the SDK.

<Tabs className="codetabs" groupId="code-examples">
<TabItem value="js" attributes={{className: "codetab js"}} default>
```typescript
import { PushChain } from '@pushchain/core'

// Define the Push testnet chain configuration
const pushTestnet = PushChain.viem.defineChain({
id: 9000,
name: 'Push Testnet',
nativeCurrency: {
decimals: 18,
name: 'nPUSH',
symbol: 'nPUSH',
},
rpcUrls: {
default: {
http: ['https://evm.pn1.dev.push.org'],
},
},
blockExplorers: {
default: {
name: 'Push Explorer',
url: 'https://explorer.dev.push.org/',
},
},
})

// Create a public client for Push testnet
const client = PushChain.viem.createPublicClient({
chain: pushTestnet,
transport: PushChain.viem.http(),
})

````

If you want to simplify the `defineChain` logic, the Push Chain SDK provides a constant named `PushChain.CONSTANTS.VIEM_PUSH_TESTNET`. You can use it to create your public client without redefining the chain configuration:

<Tabs className="codetabs" groupId="code-examples">
<TabItem value="js" attributes={{className: "codetab js"}} default>
```typescript
import { PushChain, CONSTANTS } from '@pushchain/core'

const viemClient = PushChain.viem.createPublicClient({
  chain: PushChain.CONSTANTS.VIEM_PUSH_TESTNET,
  transport: PushChain.viem.http(),
})
````

## Example

Here's an example of how to initialize the Push Chain SDK with a viem client and fetch the latest block.

<Tabs className="codetabs" groupId="code-examples">
<TabItem value="js" attributes={{className: "codetab js"}} default>
```typescript
import { CONSTANTS } from '@pushchain/core'
import { createPublicClient, http } from 'viem'

// Create a public client using the predefined testnet configuration
const client = createPublicClient({
chain: PushChain.CONSTANTS.VIEM_PUSH_TESTNET,
transport: http(),
})

// Fetch latest block
const latestBlock = await client.getBlock()
console.log('Latest block:', latestBlock)

````
