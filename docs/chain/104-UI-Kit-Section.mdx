# UI Kit

UI Kit from Push Chain allows you to seamlessly make your app compatible with all layer 1s and l2s instantly. It abstracts away the wallet connection logic, no matter from which chain you connect app your from.

UI Kit integration takes less than 5 minutes and is made for use by any React based app, no matter from which chain you connect app your from.
